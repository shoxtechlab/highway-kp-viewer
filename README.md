# Highway KP Viewer

A web application for visualizing expressway routes, KP locations, and road structures.

## Features

- Longitudinal route diagram
- KP visualization
- Structure visualization (bridges, tunnels, etc.)
- Route geometry based on OpenStreetMap data
- Street View integration

## Data Sources

Map data © OpenStreetMap contributors

## Notes

KP values are estimated from route geometry and may not exactly match official road chainage.
